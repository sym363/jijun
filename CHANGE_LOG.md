## v2.1.4.4 (2026-03-21) - 插件儲存引擎升級與安全強化
### ✨ 新功能
- **插件儲存引擎升級**: PluginStorage 從同步 `localStorage` 遷移至非同步 `IndexedDB`。現在插件資料將持久化於資料庫中，提升了存取大數據時的效能。
- **同步 API 相容層**: 透過記憶體快取 (In-Memory Cache) 實作，讓現有插件不需修改代碼即可享受非同步儲存的優點。

### 🐞 Bug Fixes
- **安全邊界強化**: 內部快取改用 `Object.create(null)`，防止插件透過 `getItem` 存取到 `toString` 或 `__proto__` 等原生屬性。
- **非同步時序修復**: 重構防抖 (Debounce) 儲存邏輯，修復在高頻率寫入時 Promise 解析可能發生的狀態衝突。

### 🔧 改進優化
- **相容性修復**: 插件 ID 驗證機制由硬性攔截降級為開發者警告，確保舊版插件能順利遷移。
